\# AgroFull - Smart Orders



\*\*Proyecto Integrado – Técnico en Analista Programador\*\*  



Este proyecto fue desarrollado como \*\*solución para la asignatura Proyecto Integrado\*\*, que funciona como un mini proyecto de título para egresar del Técnico en Analista Programador. La problemática planteada por la empresa consistía en la gestión de pedidos en una tienda online de productos agrícolas, y la solución implementada permite automatizar el resumen de pedidos en Excel, facilitando la gestión y control diario.



\## Descripción del proyecto

AgroFull es una tienda online donde los usuarios pueden:

\- Registrar y gestionar su cuenta.

\- Explorar productos agrícolas disponibles.

\- Realizar pedidos y agregarlos a un carrito.

\- Generar automáticamente un resumen diario de los pedidos en formato Excel, con totales por producto.



La aplicación está desarrollada en \*\*Django\*\*, con plantillas HTML, CSS y JS para la interfaz, y permite un manejo sencillo de los pedidos y productos para la empresa.



\## Funcionalidades principales

\- Registro y login de usuarios.

\- Gestión de productos, pedidos y administración interna.

\- Generación automática de reportes en Excel.

\- Interfaz web intuitiva y funcional.

\- Proyecto estructurado con buenas prácticas de Django.



\## Estructura del proyecto

\- \*\*AgroFull/\*\*: Configuración principal de Django.

\- \*\*applications/\*\*: Aplicaciones del proyecto (usuarios, pedidos, administración, productos, core).

\- \*\*templates/\*\*: Plantillas HTML para la interfaz.

\- \*\*static/\*\*: Archivos CSS, JS y multimedia.

\- \*\*reportes/\*\*: Archivos Excel generados con los resúmenes de pedidos.

\- \*\*manage.py\*\*: Archivo de administración de Django.



\## Cómo ejecutar el proyecto

1\. Activar el entorno virtual: `.\\venv\\Scripts\\activate`

2\. Instalar dependencias: `pip install -r requirements.txt`

3\. Ejecutar el servidor: `python manage.py runserver`

4\. Abrir en el navegador: `http://127.0.0.1:8000/`
